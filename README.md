# Strong Mind Strong Body

Minimal static website for Strong Mind Strong Body, a London-based personalised nutrition, training, recovery, and mindset coaching brand.

## Files

- `index.html` - page structure and content
- `styles.css` - visual design and responsive layout
- `script.js` - small navigation and icon setup
- `assets/` - site images

## Preview

Open `index.html` directly, or serve the folder locally with any static file server.
